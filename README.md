# Gpa-calculator

此程式使用node撰寫，執行後可以直接去南台選課系統抓取你的成績，並自動計算你的GPA成績。

## 事前準備

請先將此project下載到電腦:

    git clone https://github.com/opjlmi/gpa-calculator.git

安裝程式會使用到的相關modules:

    cd gpa-calculator
    npm install

## 執行方式

    node login.js [學號] [密碼]
